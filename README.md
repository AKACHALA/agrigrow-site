# 🌱 Agrigrow Website

Agrigrow is a digital platform designed to support farmers in Kenya and beyond by providing practical solutions to common agricultural challenges such as pests, diseases, and low productivity.

---

## 🚀 Live Website
Visit the site here:
https://your-username.github.io/agrigrow-site/

## 📌 Project Overview
This website focuses on:
- Livestock disease management
- Crop pest and disease control
- Soil and water conservation
- Smart farming practices

It is built as a simple, fast, and accessible platform for farmers.
